#Titulo
Projeto xboxlive. Faz cadastros de jogos, usuarios, perfis, generos e mostra os seus jogos favoritos e jogos que tem vinculados ao seu perfil


Clonando
git clone https://github.com/AlexFariaa/projeto-jogo

Instalação
$ npm install

Rodando o app
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

Testes
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

Construido com:
NodeJS | NestJs | TypeScript
